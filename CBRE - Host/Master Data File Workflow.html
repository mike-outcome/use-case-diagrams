Outcome Automation Pipeline (n8n)
1 Email Intake & Extraction

Receive & Buffer

n8n monitors inbox for emails from CBRE with zip attachments. Extracts CSVs from each zip. Buffers across staggered emails. Validates all 6 expected filenames are present before triggering the merge.

2 Merge & Format

Join by Channel Name & Apply CBRE Specs

Read all 6 CSVs. Join on channel name (VTS property identifier) to produce a single merged dataset with all metrics per property. Apply CBRE column naming, ordering, number formatting, and date formatting. Output as .xlsx with a "Merge" tab.

3 Historical Append

Append to Cumulative YTD Dataset

Tag current week's data with the delivery date. Append to the running historical dataset. Supports time-range queries: weekly, monthly, quarterly, annually, all-time. Interim file-based storage (v1). Database integration deferred to v2.
