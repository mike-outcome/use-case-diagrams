Transformation

Ingest & Normalize

Pull data from Armis API + file uploads into Outcome data grid. Normalize site location fields (split US_NY_NY_395NinthAve into Country / State / City). Merge with OT inventory and building context data.

Classify & Assess Risk

Classify devices into OT building categories (BMS, Access Control, Surveillance, Elevators, Lighting, Metering, etc.). Calculate vulnerability counts by severity level (Critical / High / Medium / Low). Compute total assets and percentage distribution per risk category.

Generate Report & Apply Formatting

Build HTML report with color-coded risk tables. Apply Bhruti's color-coding rules for severity thresholds. Add columns for total assets and % of assets per risk category. Generate downloadable PDF version.
