CBRE — Metering & Tenant Billbacks

Current State vs. Outcome Automation
400 North Aberdeen, Chicago — Life Sciences Building — 15+ Lab Tenants
Today — Manual Process
Trigger

Monthly Billing Cycle

Tim decides it's time to run billing. No automated trigger. Timing depends on when meter data is available and when Tim has time.

Inputs

Meter Data

CHW, HW, DW, compressed air, gas readings from the BAS. Electricity kWh from sub-meters (4 tenants). Tim collects and consolidates all readings.

Tenant Registry

15+ suites with design airflow %, CHW/HW loads, sub-meter flags. Embedded in the workbook.

Variable Utility Rates

ComEd electric, Nicor gas, city water. Tim checks if rates changed since last month.

Transformation

Tim Manually Enters Readings

Opens the HVAC Cost Workbook (603 columns). Inputs monthly readings into "Tenant Submetered Usages" for each tenant × utility type. Updates rate cells if provider pricing changed.

Excel Formulas Calculate Charges

Consumption × rate per utility. Plant costs allocated by design airflow %. Calculates per-tenant totals across 6 utility types.

Outputs

Manual Tenant Breakdown Documents

Tim creates tenant-specific documents from the spreadsheet results showing what each tenant owes. Sends to the property management team, who handles actual invoicing.

No trend analysis. No anomaly detection.
Tim eyeballs month-over-month changes. Flags spikes by memory and experience only.

With Outcome — Automated Pipeline
Trigger

Meter Data Arrives

Pipeline initiates when Tim uploads data or BAS auto-exports to Outcome.

C1 What BAS system? Can it auto-export?

Inputs

Meter Data

CHW, HW, DW, compressed air, gas, electricity readings ingested automatically from uploaded files or BAS export.

C6 Complete list of meter types and locations?

Tenant Registry

Suites, airflow %, sub-meter flags stored in Outcome Data Grid.

Variable Utility Rates

Rates with effective dates in Data Grid. Tim updates when providers change pricing.

C2 How often do rates change?

Transformation

Outcome Data Grid

Tim's Excel formulas converted to Data Grid business logic. Consumption × rate, plant cost allocation by airflow %, admin fee. Replaces the spreadsheet entirely.

C3 Admin fee %? Is one applied today?

Outputs

Tenant Breakdown Documents

Per-tenant documents with utility breakdown, consumption, rates, totals. Delivered to Tim for the property management team.

C4 Can Tim share an example of the current format?

Phase 2 — After Core Delivery

Trend Analysis: MoM & YoY consumption/cost trends per tenant per utility
Anomaly Detection: Flags unusual spikes or drops with lab-specific baselines
